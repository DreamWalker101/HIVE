Search HIVE memory for a topic.
Usage: /project:memory <topic>
Searches:
- grep -i "<topic>" ~/HIVE/memory/SESSION_INSIGHTS.md
- grep -i "<topic>" ~/HIVE/memory/prompts.log
- grep -i "<topic>" ~/HIVE/memory/findings.md
- grep -i "<topic>" ~/.hermes/workspace/AGENTS.md
Return the most relevant 10-20 lines found.
