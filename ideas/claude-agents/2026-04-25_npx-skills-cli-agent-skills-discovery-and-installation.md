## Why It Matters
Ahmed already uses skills extensively (the superpowers stack, gsd, etc.) but these were installed manually. The `npx skills` CLI provides a standardized discovery and update mechanism — meaning there are likely high-quality skills for common tasks (React, testing, deployment, PR review) that can be found and installed in seconds rather than built from scratch.

## How To Use It

```bash
# Search for a skill
npx skills find react performance
npx skills find pr review
npx skills find changelog

# Install a skill globally (no prompts)
npx skills add vercel-labs/agent-skills@react-best-practices -g -y

# Check for updates on installed skills
npx skills check

# Update all installed skills
npx skills update
```

**Discovery workflow:**
1. Check https://skills.sh/ leaderboard first — popular skills for major domains are listed there
2. If not found, run `npx skills find [query]` with specific keywords
3. Try alternative terms if first search fails ("deploy" → "deployment" or "ci-cd")
