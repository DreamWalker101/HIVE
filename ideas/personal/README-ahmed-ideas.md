# Ahmed's Ideas Vault

Personal project concepts, build ideas, and sparks of inspiration.

These are seeds — rough thinking captured from reels, articles, and conversations.
Not fact-checked. Not polished. Just preserved.

## Status Tags
- `seed` — just captured, not yet explored
- `exploring` — actively thinking about it
- `parked` — good idea, not the right time
- `shipped` — built or implemented

## Structure
Ideas are captured as individual markdown files, one idea per file.
Each file preserves the original note and expands it into actionable next steps.
