# Ideas Vault

Personal project concepts and build ideas captured from AI/tech content.

These are seeds — rough thinking sparked by reels, articles, or conversations.
They are NOT fact-checked; they are preserved for future exploration.

## Status Tags
- `seed` — just captured, not yet explored
- `exploring` — actively thinking about it
- `parked` — good idea, not the right time
- `shipped` — built or implemented

## Format
Each file has Ahmed's original note front and centre, with context from the source content.
