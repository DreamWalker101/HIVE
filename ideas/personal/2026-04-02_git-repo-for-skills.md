# Git Repo for Skills — A Personal Claude Skills Library

## The Idea
Maintain a personal git repo of Claude Code skills, slash commands, and role-based personalities — similar to how Gary Tan published his GStack repo. Instead of rebuilding prompts and setups from scratch, everything lives in version control and can be cloned, forked, or updated. This becomes Ahmed's own curated toolkit for AI-assisted work, tailored to his actual workflows.

## What Sparked It
Gary Tan (CEO of Y Combinator) published a repo called GStack with 31 skills, 23 specialized personalities, and power tools for browser control, safety, and multi-AI reviews. The key insight: "instead of using one generic AI, you can call role-based experts" — product strategists, engineering managers, security officers, all as callable skills.

## Why This Could Matter
A versioned skills repo means expertise is composable and reusable — no more re-prompting from memory. It also makes the setup portable across machines and shareable with collaborators.
