# Takumi: JSX-to-Image/Animation Renderer with Dedicated LLM Documentation

## What Is This
Takumi (https://takumi.kane.tw) is an open-source library that renders React/JSX components into static images, animations, and video frames at native speed. It supports rich CSS layout, WOFF2 fonts, and complex text scripts. The notable detail: it ships with a dedicated **"For LLMs"** documentation tab in its navbar — meaning its API and capabilities are explicitly formatted for LLM agent consumption.

## Why It Matters
Most JSX-to-image tools (Satori, html-to-image) are designed for human developers. Takumi explicitly targets LLM workflows with structured docs. This makes it a strong candidate for AI agents that need to generate visual assets (OG images, report cards, screenshots, video thumbnails) from programmatic JSX — without a browser.
