# Claude Code Playground Skill: Visual Design Builder with Prompt Export

## What Is This
The Playground is an official Anthropic Claude Code plugin that turns Claude into an interactive visual design tool. When invoked, Claude generates a self-contained HTML file with sliders, dropdowns, color pickers, and presets — letting you tweak typography, spacing, colors, and layout visually. Once satisfied, you copy the generated natural-language prompt and paste it back into Claude Code to implement the changes.

## Why It Matters
Iterating on visual design via text prompts alone is tedious and imprecise — you describe a change, wait for Claude to implement it, review, then repeat. The Playground breaks that loop: you dial in the exact design you want with your eyes first, then give Claude one comprehensive prompt. It's the difference between "a bit more padding, warmer tone, bigger headings..." (5 rounds) vs. one precise prompt generated from your visual choices (1 round).

## How To Use It

### Install the Playground Plugin
```
# Inside Claude Code CLI (not Desktop — /plugin is CLI-only):
/plugin
# → Go to Discover tab
# → Search "playground"
# → Install from official Anthropic marketplace (claude-plugins-official)

# Or install directly:
/plugin install playground@claude-plugins-official
```

After installing, restart Claude Code or run `/reload-plugins` (note: hyphenated) to activate it.

### Invoke the Skill
Once installed, ask Claude to build a playground for your site/component:
```
Use the playground skill to let me visually edit my website's design and color scheme
```

Claude will generate a self-contained HTML file that opens in your browser with:
- Live preview of your site/component
- Controls panel: color pickers, font selectors, spacing sliders, size inputs
- Prompt output box at the bottom with a copy button

### Workflow
1. Open the generated HTML file in a browser
2. Adjust controls until the design looks right
3. Click "Copy Prompt" — this gives you a natural-language description of all your choices
4. Paste that prompt back into Claude Code to implement the actual changes
