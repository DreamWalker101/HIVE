# Fireworks Tech Graph: Claude Code Skill for Natural Language → SVG/PNG Technical Diagrams

## What Is This
`fireworks-tech-graph` is a Claude Code skill (installed via SKILL.md) that converts natural language descriptions into publication-ready SVG and PNG technical diagrams. It has deep built-in knowledge of AI/Agent domain patterns (RAG, multi-agent flows, tool calls, Mem0) and supports 8 diagram types with 5+ visual styles. SVG is generated first, then exported to high-res PNG via `rsvg-convert`.

## Why It Matters
If you use Claude Code and need architecture diagrams, system flow charts, or AI pipeline visuals, this gives you a zero-friction path: describe what you want in plain English, get a polished image. It's faster than Excalidraw-style tools when you don't need interactive editing — no drag-and-drop, just describe and render. The AI/Agent semantic shape library (hexagon = agent, cylinder = storage, double-border rect = LLM) makes it particularly useful for the kind of systems Ahmed builds.

## How To Use It

1. **Install the skill** — copy the `SKILL.md` from the repo into your project or Claude Code skills directory:
   ```bash
   # Clone or download SKILL.md from:
   # https://github.com/yizhiyanhua-ai/fireworks-tech-graph
   ```

2. **Invoke in Claude Code** — describe what you want:
   ```
   /diagram Draw a RAG pipeline with a vector store, embedding model, and LLM responder. Use the aesthetic style.
   ```

3. **Output** — Claude generates SVG inline, then converts to PNG:
   ```bash
   rsvg-convert -w 1200 diagram.svg -o diagram.png
   ```
   (Requires `librsvg2-bin` on Linux: `sudo apt install librsvg2-bin`)

4. **Supported diagram types**: system architecture, sequence, flowchart, UML (all 14 types), AI/Agent pipelines, network topology, ER diagrams, mind maps

5. **Visual styles**: aesthetic, minimal, technical, dark, blueprint — specify in your prompt
