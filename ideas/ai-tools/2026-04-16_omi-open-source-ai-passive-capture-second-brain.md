# OMI: Open-Source Passive Capture AI Second Brain (Screen + Audio + Wearable)

## What Is This

OMI (by BasedHardware) is a fully open-source AI memory layer that passively captures your screen and conversations in real-time, then gives you searchable history, summaries, and action items across everything you've seen and heard. It runs on desktop, mobile, and a dedicated wearable device ($89), making it a continuous ambient capture system rather than a point-in-time tool.

## Why It Matters

Most AI tools are reactive — you have to initiate them. OMI is ambient: it runs continuously and retrospectively answers questions like "what did we decide in that meeting last Tuesday?" or "what was that GitHub repo someone mentioned?". For someone running heavy agentic workflows (like Ahmed's brain/body setup), OMI functions as the persistent episodic memory layer that Claude sessions currently lack between conversations.

- Always-on mic; says "Hey Omi" to activate assistant

**Key capabilities to use:**
- Ask it to summarize the last N hours of screen/audio
- Query across captured history ("what was that tool mentioned in the meeting?")
- Use the open plugin SDK to pipe OMI's memory into your own agents or Claude sessions
- Self-host the backend to keep data local (privacy-safe option)

**Integration angle for Claude workflows:**
OMI's open plugin system means you can expose a captured-memory API endpoint and call it from Claude tools — effectively giving Claude persistent episodic context from your day without manually copy-pasting meeting notes.
