# Agent Council: Multi-Perspective Claude Skill to Counter Sycophancy

## What Is This

The Agent Council is a Claude Code skill that spins up **5 adversarial AI advisors**, each approaching your question from a distinct cognitive angle. Their answers are anonymized, cross-reviewed by peer agents, then synthesized by a "chairman" agent into one verdict and one clear next step. It directly addresses Claude's documented sycophancy problem by structurally preventing any single perspective from dominating.

## Why It Matters

Claude (and all RLHF-trained models) are demonstrably sycophantic — Anthropic's own research confirms models prefer responses that match user views even when wrong. Asking Claude "is this a good idea?" will yield a supportive answer; asking "why is this bad?" yields criticism. The Council pattern breaks this by forcing multiple contradictory framings in parallel before synthesis, so the output reflects structured disagreement rather than what you want to hear. This makes it genuinely useful for architecture decisions, feature prioritization, and strategy validation.

**Trigger inside Claude Code:**
```

```

**The 5 advisor roles it instantiates:**
| Advisor | Role |
|---|---|
| **Contrarian** | Hunts for fatal flaws — assumes it will fail |
| **First Principles** | Ignores your framing — asks what you're actually trying to solve |
| **Expansionist** | Hunts for upside you're missing |
| **Outsider** | Gets zero context about you — catches obvious blind spots |
| **Executor** | Only cares about what happens next — what do you do Monday morning? |

**Synthesis pipeline:**
1. All 5 advisors answer independently
2. Answers are shuffled and anonymized
3. 5 peer reviewer agents cross-review all answers
4. A chairman agent synthesizes into one verdict + one next step

**Best applied to:**
- "Should I build feature X?"
- "Is this architecture the right call?"
- "What should I prioritize next?"
- Any decision where you suspect you already have a preferred answer
